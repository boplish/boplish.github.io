Class: Peer

Peer

Represents a foreign Peer. Used by Router instances to route messages from one Peer to another. Control over a Peer lies in the hand of a ConnectionManager instance.

new Peer(id, peerConnection, dataChannel)

Parameters:
Name Type Description
id String
peerConnection RTCPeerConnection The PeerConnection to the remote peer.
dataChannel DataChannel The DataChannel to the remote peer.
